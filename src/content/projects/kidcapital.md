A board game app that teaches children financial literacy. Monopoly-inspired mechanics with educational goals. Mobile-first with Capacitor for iOS and Android.

KidCapital takes the engaging loop of board games: roll dice, move, make decisions: and fills it with real financial concepts. Kids learn about saving, investing, budgeting, and compound interest without realizing they're in a lesson. Each game session is 15-20 minutes, designed for attention spans that won't sit through a 2-hour Monopoly marathon.

Built with React and TypeScript, packaged with Capacitor for native mobile deployment. The game state is entirely client-side: no server needed, no accounts required. Parents can track progress through a simple dashboard that shows which financial concepts their kids have encountered and mastered.
